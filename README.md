# 🧮 Simple C++ Calculator

A Simple desktop calculator application built with **C++17** and **wxWidgets**.

## ✨ Features

* **Basic Operations:** Addition, Subtraction, Multiplication, and Division.
* **Order of Operations:** Respects mathematical priorities (DMAS).
* **Error Handling:** Protects against division by zero and invalid operators.
* **Modern UI:** Responsive layout that scales with window resizing.

## 🛠️ Build Requirements

* **C++ Compiler:** Supports C++17.
* **Build System:** CMake (3.16+).
* **Framework:** wxWidgets.

## 🚀 How to Build (on Linux)

1. **Clone the project:**

```bash
git clone https://github.com/yahia-mo/cpp-GUI-Calculatoir.git
cd cpp-GUI-Calculatoir
   ```

1. **Build using CMake::**

```bash
mkdir build 
cd build
cmake ..
cmake --build .
``` 

3. **Run the application:**

```bash
./calc
```
